# pymusic
A simple lightweight music player written in python using ttkbootstrap

![screenshot](https://i.ibb.co/ct45vZ9/pymusic.png))

## Current Version 
V1.0.0-Beta


### Getting Started:

  **Install packages**
  ```sh
pip install -r requirements.txt
```
  **Run program**
   ```sh
   python pymusic.py
   ```
